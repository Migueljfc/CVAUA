Ligação do servidor

1. Ligar o Desktop Aurora que se encontra no local indicado na imagem.

2. Ligar os monitores da videowall usando o comando da NEC.

3. Após o windows iniciar abrir o ficheiro SAGE2.bat.

4. Após isso irá ser iniciado o servidor no navegador predefinido no nosso caso o Chrome. Para o servidor inicial terá de colocar as seguintes credencias: "user": sage2  "password": sage2 .

5. Após preencher as credenciais corretamente irá aparecer a interface de controlo do SAGE2.

a. Se pretende definir uma password para alguém se conectar ao servidor da videowall terá que preencher o campo respetivo e clicar em "Set Meeting Password", se não pretender definir uma password passe para o passo 4.b.

b. Para iniciar o servidor basta clicar no botão "Start " .

nota: É de salientar que todos os passos apresentados anteriormente são realizados no Desktop Aurora

Desligar o servidor

1. Abra a janela da interface de controlo do SAGE2 para isso prima Alt + tab  pois com a iniciação do servidor o rato do desktop fica inoperacional.

 2. Ao voltar à interface de controlo do SAGE2 basta clicar em "Stop" .
