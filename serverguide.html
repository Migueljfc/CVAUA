Ligação do Servidor

Ligação do servidor

Para ligar o servidor tem que ligar o Desktop Aurora correr o ficheiro SAGE.bat, deve preencher ambas as credenciais com sage2 e após aparecer a interface de controlo clicar no botão Start. 

Configuração do servidor

Para configurar o servidor tem de escolher a opção Admin na interface de controlo do SAGE2.

Ligação do servidor

1. Ligar o Desktop Aurora que se encontra no local indicado na imagem.

2. Após o windows iniciar abrir o ficheiro SAGE2.bat.

3. Após isso irá ser iniciado o servidor no navegador predefinido no nosso caso o Chrome. Para o servidor inicial terá de colocar as seguintes credencias: "user": sage2  "password": sage2 .

4. Após preencher as credenciais corretamente irá aparecer a interface de controlo do SAGE2.

a. Se pretende definir uma password para alguém se conectar ao servidor da videowall terá que preencher o campo respetivo e clicar em "Set Password", se não pretender definir uma password passe para o passo 4.b.

b. Para iniciar o servidor basta clicar no botão "Start " .

nota: É de salientar que todos os passos apresentados anteriormente são realizados no Desktop Aurora

Enviar conteúdo para o servidor

1. Via drag and drop de ficheiros no formato png, jpg, pdf e mp4, para isso basta escolher o ficheiro do seu computador nos formatos indicados a cima e arrastar para a página

 2. Via partilha de ecrã, é possível partilhar o ecrã na sua totalidade ou uma janela específica.

a. Clicar no ícone com formato de um monitor

b. Após aparecer a janela escolha a aplicação ou ecrã que pretende partilhar

3. Pelas aplicações nativas do SAGE2

a. Clicando no ícone em formato de foguetão no menu inicial do SAGE2

b. Diretamente na VideoWall através do Ponteiro clicando com o botão do lado direito do rato

Nota: Para saber mais sobre o Ponteiro visitar as secções seguintes
